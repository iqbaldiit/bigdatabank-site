Hadoop

The foundational framework that started the big data revolution.

What it is

An open-source framework that allows for the distributed processing of large data sets across clusters of computers using simple programming models (MapReduce) and distributed storage (HDFS).

Why learn this?

While cloud-native tools are taking over, many large enterprises still run on-premise Hadoop clusters. Understanding HDFS and YARN gives you a deep understanding of distributed systems.

How it fits (Architecture)

YARN (Compute)
HDFS (Storage)
